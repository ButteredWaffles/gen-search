What is Gensearch?

Gensearch is an API that allows you to access information about content in the game Monster Hunter Generations.

It features information on monster stats, armor sets, weapons, hunter arts, items, quests, skills, and palico data!
How do I use this? Documentation can be found here. There are no tokens or other authorization requirements, so no jumping through hoops to get access to what you want!
