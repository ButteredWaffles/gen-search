What is Gensearch? How do I use this? Acknowledgements

Gensearch is a RESTful API that allows you to access information about content in the game Monster Hunter Generations.

It features information on monster stats, armor sets, weapons, hunter arts, items, quests, skills, and palico data!

 Documentation can be found here. There are no tokens or other authorization requirements, so no jumping through other hoops to get access to what you want!

Many thanks to Kiranico for gathering and centralizing the information.
